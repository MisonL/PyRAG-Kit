# 用户指南

本指南将帮助您了解如何运行 PyRAG-Kit 的主菜单、聊天会话和召回测试。

## 1. 启动程序

当您按照 [快速上手](./getting-started.md) 指南成功运行 `uv run main.py` 后，程序会进入主菜单。

您会看到类似以下的选项：

```
1. 知识库文档向量化处理
2. 召回测试
3. 启动聊天机器人会话
4. 退出程序
```

## 2. 菜单项说明

### 2.1 知识库文档向量化处理

该菜单项会处理 `knowledge_base/` 目录中的 Markdown 文件，生成新的知识快照，并把活动快照指针更新到最新构建结果。

如果您只想单独执行这一步，也可以运行：

```bash
uv run python -m scripts.embed_knowledge_base --mode standard
```

### 2.2 召回测试

选择该项后，程序会进入检索测试模式。输入查询后，会显示命中的文档来源、内容预览和得分。

输入 `/quit` 可退出召回测试并返回主菜单。

### 2.3 聊天会话

选择该项后，程序会加载当前活动知识快照并进入聊天界面。

程序会执行以下步骤：

1. **意图识别**：先提炼用户问题的核心检索意图。
2. **知识库检索**：根据识别后的意图执行检索，并显示检索详情。
3. **生成回答**：结合检索结果和原始问题生成最终回答。

## 3. 特殊命令

在聊天过程中，您可以随时输入以下特殊命令来控制程序：

### `/config` - 打开配置菜单

这是 PyRAG-Kit 最强大的功能之一。输入 `/config` 会暂停聊天，并进入一个交互式配置菜单。

在菜单中，您可以使用键盘方向键进行选择，并按回车确认。主要功能包括：

*   **调整检索参数**:
    *   **检索模式**: 在“全文检索”、“向量检索”和“混合检索”之间切换。
    *   **检索数量 Top K**: 调整每次查询返回的文档数量。
    *   **Rerank重排**: 启用或禁用 Rerank 模型进行二次精排。
    *   **混合搜索权重**: 当使用“混合检索”时，调整向量和关键词搜索结果的权重。
*   **切换模型**:
    *   **语言模型 (LLM)**: 动态切换当前对话使用的聊天模型（例如，从 `google` 切换到 `openai-gpt4o`）。切换后，程序会自动重载模型。
    *   **重排模型 (Rerank)**: 切换用于精排的 Rerank 模型。
*   **查看当前完整配置**: 显示所有当前生效的聊天和模型参数。

完成配置后，选择“保存并返回聊天”即可应用更改并继续对话。

### `/quit` - 退出当前会话

在聊天会话或召回测试中输入 `/quit` 可以返回主菜单。您也可以使用 `Ctrl+C` 或 `Ctrl+D` 快捷键退出当前交互。

## 4. 查看日志

程序运行期间的所有重要操作和聊天记录都会被记录下来。您可以在 `data/logs/` 目录下找到日志文件。

*   `chat_log_YYYY-MM-DD.log`: 记录聊天对话历史，包括用户问题、识别出的意图、命中文档数量和模型回答。
*   `app_log_YYYY-MM-DD.log`: 记录通用运行日志，包括向量化、检索、配置切换和错误信息。
